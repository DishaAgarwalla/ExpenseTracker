# 💰 Expense Tracker

A modern and scalable Expense Tracker built using React and Vite.  
This application helps users manage their daily expenses, categorize spending, and visualize financial data through charts.

## 🚀 Features

- Add, edit, and delete expenses
- Category-based expense tracking
- Expense filtering
- Summary dashboard
- Pie chart visualization
- Local storage persistence
- Clean and modular folder structure
- Context API for state management
- Custom hooks for reusable logic

## 🛠 Tech Stack

- React (Vite)
- Context API
- Custom Hooks
- Chart.js
- CSS
- Local Storage

## 📂 Project Structure

- Components (Auth, Dashboard, Charts)
- Context (Auth, Theme, Expense)
- Hooks
- Utils
- Services

## 🎯 Purpose

This project demonstrates advanced React concepts including:
- Component architecture
- Global state management
- Custom hooks
- Code modularization
- Reusable utilities
